## glab issue subscribe

Subscribe to an issue

```
glab issue subscribe <id> [flags]
```

### Examples

```
$ glab issue subscribe 123
$ glab issue sub 123
$ glab issue subscribe https://gitlab.com/profclems/glab/-/issues/123

```
