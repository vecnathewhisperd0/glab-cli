## glab issue unsubscribe

Unsubscribe to an issue

```
glab issue unsubscribe <id> [flags]
```

### Examples

```
$ glab issue unsubscribe 123
$ glab issue unsub 123
$ glab issue unsubscribe https://gitlab.com/profclems/glab/-/issues/123

```
