## glab issue reopen

Reopen a closed issue

```
glab issue reopen <id> [flags]
```

### Examples

```
$ glab issue reopen 123
$ glab issue open 123
$ glab issue reopen https://gitlab.com/profclems/glab/-/issues/123

```
