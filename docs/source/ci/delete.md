## glab ci delete

Delete a CI pipeline

```
glab ci delete <id> [flags]
```

### Examples

```
$ glab ci delete 34
$ glab ci delete 12,34,2

```

### Options

```
  -s, --status string   delete pipelines by status: {running|pending|success|failed|canceled|skipped|created|manual}
```
