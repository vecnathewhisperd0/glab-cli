## glab ci lint

Checks if your .gitlab-ci.yml file is valid.

```
glab ci lint [flags]
```

### Examples

```
$ glab ci lint  
#=> Uses .gitlab-ci.yml in the current directory

$ glab ci lint .gitlab-ci.yml

$ glab ci lint path/to/.gitlab-ci.yml

```
