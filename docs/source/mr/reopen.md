## glab mr reopen

Reopen merge requests

```
glab mr reopen [<id>... | <branch>...] [flags]
```

### Examples

```
$ glab mr reopen 123
$ glab mr reopen 123 456 789
$ glab mr reopen branch-1 branch-2
$ glab mr reopen  # use checked out branch

```
