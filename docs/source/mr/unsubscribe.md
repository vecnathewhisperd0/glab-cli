## glab mr unsubscribe

Unsubscribe from merge requests

```
glab mr unsubscribe [<id> | <branch>] [flags]
```

### Examples

```
$ glab mr unsubscribe 123
$ glab mr unsub 123
$ glab mr unsubscribe branch
$ glab mr unsubscribe 123 branch  # unsubscribe from multiple MRs

```
